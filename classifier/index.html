V1/V2 학습 파이프라인
V1 Training (PyTorch Lightning)
  • Module: ImageLightningModule
  • Loss: BCEWithLogitsLoss (multi-label)
  • Optimizer: AdamW (lr=4e-5, wd=0.01)
  • Scheduler: Cosine / Linear / MultiStep
  • Precision: FP16
  • Grad Accum: 4 steps
  • DDP: Multi-GPU 지원
  • Callbacks: Checkpoint, EarlyStopping
V2 Training (Accelerate + Hydra)
  • Module: LXDonutParallelClassificationModel
  • Loss: CrossEntropyLoss (auto-regressive)
  • Optimizer: AdamW (lr=5e-5, wd=0.01)
  • Scheduler: CosineAnnealingWarmup
  • Precision: BF16
  • Grad Clip: 1.0
  • DeepSpeed: ZeRO Stage 2
  • Calibration: Temperature Scaling

Inference Pipeline
추론 실행 흐름
# V1 Inference Flow (MultiTagDocumentClassifierPipeline)
1. Image 로드 → load_images_from_binary(PDF/PNG/TIFF)
2. Preprocess  → Resize + ImageNet Normalize
3. Forward    → Model(image) → logits [B, num_classes]
4. Sigmoid    → logits → probabilities [0, 1]
5. Threshold  → per-class threshold 적용 (threshold.yaml)
6. Priority   → 충돌 예측 해결 (priority.yaml)
7. Output     → {result: ["class1"], confidence_scores: [...], rot90: [0]}

# V2 Inference Flow (LXDonutParallelClassificationPipeline)
1. Image + OCR 로드
2. Preprocess  → Image tensor + OCR tokenize + Prompt 생성
3. Encode     → Swin(img) + BART(ocr) → LXMERT(fusion)
4. Decode     → mBART auto-regressive 생성
5. Parse      → output tokens → class name string
6. Output     → UFP format
